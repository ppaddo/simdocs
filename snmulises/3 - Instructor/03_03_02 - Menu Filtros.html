Menú Filtros

El menú Filtros consta de las siguientes opciones:

  • Carta: Desde esta opción se puede seleccionar la presentación de distintos elementos de las cartas:

    • Marcos: Mientras esta opción esté seleccionada el sistema presenta sobre la carta actual los marcos de las cartas que están situadas geográficamente dentro de ella.

    • Nombres: Mientras esta opción esté seleccionada, el sistema presenta sobre la carta actual los nombres de las cartas que están situadas geográficamente dentro de ella. Esta opción sólo puede seleccionarse cuando lo esté también la opción anterior.

    • Textos: Mientras esta opción esté seleccionada, el sistema presenta sobre la carta las etiquetas asignadas a los objetos. Evidentemente, para que se presente esta etiqueta es condición imprescindible, aparte de seleccionar esta opción, que el objeto tenga asignada una etiqueta en la carta y que la opción del menú Filtros - Nivel de detalle permita la presentación de dicho objeto.

    • Filtro según escala: Mientras esté seleccionada está opción, el sistema presentará los distintos elementos de las cartas en función de la escala seleccionada.

  • Nivel de detalle: Desde esta opción se pueden solicitar los siguientes niveles de filtro:

    • Bajo: Al seleccionar esta opción se aplica el filtro definido por ECDIS a la presentación cartográfica para mostrar el display base. Este display base se define como el nivel mínimo de información náutica que no debe ser nunca eliminada de la presentación, y consiste en la información que se requiere para navegar en todo momento, en todas las áreas geográficas y en todas las circunstancias, sin pretender que sea suficiente para una navegación segura.

    • Medio: Al seleccionar esta opción se aplica el filtro definido por ECDIS a la presentación cartográfica para mostrar el display standard. Este display standard se define como el nivel de información náutica que habitualmente es suficiente para la navegación.

    • Alto: Al seleccionar esta opción no se aplica filtro alguno a la presentación cartográfica.

      • Nota: El menú Filtros no aparece cuando se usa cartografía C-Map.
