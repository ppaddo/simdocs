Autorizaciones

Mediante esta tabla se parametrizan las Autorizaciones ROE. Una autorización (suceso autorizado) consiste en autorizar / prohibir / obligar a una Unidad a efectuar una Acción sobre otra unidad (Blanco).

Tipo: Este parámetro determina el tipo de autorización. Puede tomar los siguientes valores:

  • AUTORIZACIÓN
  • PROHIBICIÓN
  • OBLIGACIÓN

Las Unidades que a las que aplica la autorización son especificadas mediante filtros por Unidad Afectada, Bando y Tipo:

Unidad Afectada: Este parámetro determina si la unidad a la que aplica la autorización es aquella sobre la que se efectúa la acción del condicionante (Blanco) asociado, o cualquiera. Puede tomar los siguientes valores:

  • UNIDAD
  • CUALQUIERA

Bando: Este parámetro determina el bando al que deben pertenecer las unidades para las que aplica la autorización. Sólo puede ser introducido si Unidad Afectada es CUALQUIERA. Puede tomar los siguientes valores:

  • NO APLICA
  • CONTRARIO
  • PROPIO
  • NEUTRAL
  • CONTRARIO-NEUTRAL
  • PROPIO-NEUTRAL

Tipo: Este parámetro determina el tipo de unidad del que deben ser las unidades para que aplica la autorización. Sólo puede ser introducido si Unidad Afectada es CUALQUIERA. Puede tomar los siguientes valores:

  • NO APLICA
  • SUPERFICIE
  • SUBMARINO
  • AVIÓN
  • HELICÓPTERO
  • TERRESTRE
  • BASE
  • AVIÓN-HELICÓPTERO
  • SUPERFICIE-SUBMARINO
  • TERRESTRE-BASE

Las Acciones a las que autoriza / prohíbe / obliga la autorización son especificadas parametrizando:

  • Ubicación Geográfica, para especificar que las unidades tienen autorización / prohibición / obligación de permanecer o no en una zona o ruta.

  • Empleo de Sistemas, para especificar que las unidades tienen autorización / prohibición / obligación de emplear algún sistema contra blancos.

  • Condiciones Cinemáticas que las unidades tienen autorizadas / prohibidas / obligadas con respecto a blancos.

Los Blancos contra los que se efectúa la Acción, se especifican parametrizando:

  • Filtros para Blancos

Ubicación Geográfica:

Situación: Este parámetro se determina la situación sobre la Zona ROE asociada, a la que las unidades están autorizadas / prohibidas / obligadas. Puede tomar los siguientes valores:

  • NO APLICA
  • DENTRO
  • FUERA

Zona: Este parámetro determina la zona geográfica sobre la que se comprueba la Situación de las unidades. Sólo puede ser introducido si Situación es DENTRO o FUERA. Podrán ser seleccionadas de entre la Zonas ROE definidas. Mediante el botón de etiqueta “...” se abre el diálogo para definir estas zonas. Ver apartado Preparación – Modelos Básicos – ROE – Zonas ROE.

Empleo de Sistemas:

Sistema: Este parámetro determina el sistema a cuyo empleo las unidades están autorizadas / prohibidas / obligadas. Puede tomar los siguientes valores:

Sensores
  • RADAR SUPERFICIE
  • RADAR AÉREO
  • RADAR DT

  • LOCKON
  • LOCKON AÉREO
  • LOCKON SUPERFICIE
  • LOCKON SEEKER

  • SONAR HULL
  • SONAR VDS
  • SONAR TACTAS
 Contramedidas
  • SEÑUELOS

  • ECM NAV
  • ECM EXP
  • ECM DT
  • ECM SEEKER

  • COMSM HF
  • COMSM VHF
  • COMSM UHF

  • CHAFF D
  • CHAFF C
  • CHAFF S
  • CHAFF R
 Armas
  • TORPEDO

  • MISIL SAM / AAM / UAM
  • MISIL SSM / ASM / USM

  • CAÑONES

Condiciones Cinemáticas:

Distancia <: Este parámetro determina el límite de distancia de la unidad al blanco por debajo del cual la unidad está autorizada / prohibida / obligada a permanecer.

    Unidades: yardas

    Rango: 0 – 999999

CPA <: Este parámetro determina el límite de CPA de la unidad sobre el blanco por debajo del cual la unidad está autorizada / prohibida / obligada a permanecer.

    Unidades: yardas

    Rango: 0 – 999999

Altura <: Este parámetro determina el límite de altura de la unidad por debajo del cual la unidad está autorizada / prohibida / obligada a permanecer.

    Unidades: pies

    Rango: 0 – 99999

Velocidad <: Este parámetro determina el límite de velocidad de la unidad por debajo del cual la unidad está autorizada / prohibida / obligada a permanecer.

    Unidades: nudos

    Rango: 0 – 9999,9

Velocidad >: Este parámetro determina el límite de velocidad de la unidad por encima del cual la unidad está autorizada / prohibida / obligada a permanecer.

    Unidades: nudos

    Rango: 0 – 9999,9

Filtros para Blancos:

Blanco Afectada: Este parámetro determina si el blanco sobre el que la unidad está autorizada / prohibida / obligada a efectuar la acción es la unidad que activa el condicionante asociado, o cualquiera. Puede tomar los siguientes valores:

  • UNIDAD
  • CUALQUIERA

Bando: Este parámetro determina el bando al que deben pertenecer el blanco sobre el que la unidad está autorizada / prohibida / obligada a efectuar la acción. Sólo es tenido en cuenta si Unidad Afectada es CUALQUIERA. Puede tomar los siguientes valores:

  • NO APLICA
  • CONTRARIO
  • PROPIO
  • NEUTRAL
  • CONTRARIO-NEUTRAL
  • PROPIO-NEUTRAL

Tipo: Este parámetro determina el tipo de unidad del que deben ser los blancos el blanco sobre el que la unidad está autorizada / prohibida / obligada a efectuar la acción. Sólo es tenido en cuenta si Unidad Afectada es CUALQUIERA. Puede tomar los siguientes valores:

  • NO APLICA
  • SUPERFICIE
  • SUBMARINO
  • AVIÓN
  • HELICÓPTERO
  • TERRESTRE
  • BASE
  • AVIÓN-HELICÓPTERO
  • SUPERFICIE-SUBMARINO
  • TERRESTRE-BASE
