Recuperación del Sistema

Instalación de Preparación y Consola de Galeón en PC:

Para la instalación de la Consola del Simulador Táctico Galeón, se entrega un único fichero comprimido cuyo nombre identifica la ubicación para la que ha sido generada:

Galeon.SITE.Install.exe

  • Como primer paso, ejecutaremos este fichero para instalarlo. Por defecto se instala en el directorio Galeon.master.
  • Segundo paso: copiar el contenido de Galeon.master en C:\Archivos de programa\Galeon.
  • Los siguientes pasos deben realizar sólo si se instala por primera vez:
    • Establecer en Configuración Regional del sistema el punto decimal como Símbolo Decimal.
    • Establecer la variable de entorno CONSOLE_NUMBER en variables de entorno del sistema, asignándola al número de consola.
    • Ir al directorio de fuentes de Windows y hacer doble click sobre la fuente Digiface.
    • Instalación del cliente de Base de Datos Oracle, Informix u ODBC (ver más adelante). Este paso sólo es necesario realizarlo si en el PC en el que se está instalando se desea ejecutar también la Preparación o si se desea que la Consola pueda ser ejecutada el Modo Local.

Nota 1: La configuración típica del Simulador Táctico Galeón se compone de un servidor y un número más o menos elevado de Consolas que actúan como puestos de Alumno o Instructor. Con esta configuración, en la que hay que instalar (ya sea por primera vez o instalación de nuevas versiones) la Consola de Galeón en un alto número de PCs, puede ser muy útil el uso de algún software de réplica de ficheros (por ejemplo, Second Copy 2000) para automatizar el segundo paso, es decir, para copiar el contenido de Galeon.master en C:\Archivos de programa\Galeon.

Nota 2: Hay que tener en cuenta que si se está instalando en un PC que sólo dispone un monitor se debe cambiar en el directorio personal del usuario, en el fichero Galeon.rc los parámetros BorderAndTitle = 1 y ACWy = 0.

Instalación de Cliente Oracle:

Basta con definir las siguientes variables de entorno del sistema:

ORACLE_HOME = C:\Archivos de programa\Galeon\DB.Drivers\Oracle

Comprobar que el siguiente fichero es correcto y que los nombres de hosts que aparecen son accesibles: C:\Archivos de programa\Galeon\DB.Drivers\Oracle\network\admin\tsnames.ora, según la instalación realizada previamente y accesible por red del servidor de base de datos Oracle.

Instalación de Cliente Informix:

Ejecutar setnet32.exe, e introducir la siguiente configuración:

INFORMIXDIR = C:\Archivos de programa\Galeon\DB.Drivers\Informix
INFORMIXSERVER = galserver_on
SERVER INFORMATION:
Informix server: galserver_on
Hostname: galserver
Protocolname: onsoctcp
ServiceName : 1526

Pulsar Make DEFAULT SERVER

En esta configuración se ha supuesto que el servidor de base de datos instalado y accesible por la red se la ha denominado galserver_on. Si no es así, sustituir este nombre por el correcto. Debe configurarse correctamente el usuario y passwd de la base de datos sysmaster en Galeon.rc

Instalación de Cliente ODBC:

En Panel de control ® Herramientas Administrativas ® Origenes de datos ODBC, agregar nuevo DSN de Sistema asignándole controlador Microsoft Access Driver, y estableciendo como nombre del DSN ds_galeon.

Debe configurarse el parámetro USEDBLINK=1 en Galeon.rc.
