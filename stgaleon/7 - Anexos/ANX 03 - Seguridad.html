Seguridad

Los niveles de seguridad del sistema son:

  • Nivel de Documentación, en este caso los datos que se consideren reservados en cualquier documento, pasarán a un anexo con la clasificación correspondiente.
  • Nivel de Base de Datos, todos los modelos básicos de la Base de Datos tendrán un nivel de seguridad especificado en la introducción del mismo.
  • Nivel de Ejercicio / Unidad, cada ejercicio lanzado tendrá un nivel de seguridad concreto, y cada unidad del mismo también. Estos niveles de seguridad impedirán ver durante el ejercicio la información que se ha considerado clasificada.
    Afecta a Consultar BD y a los modelos de ESM, ECM y COMSM para determinar si muestran información de los equipos detectados.
  • Nivel de Sistema Informático, mediante “passwords” y capacidades de cada usuario del sistema.
