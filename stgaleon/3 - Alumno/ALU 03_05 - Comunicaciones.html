Comunicaciones

Superficie

El área Satélite (SAT) muestra el estado del transceptor satelital.

El área Data Link (D.Link) muestra el estado del equipo de Data Link, así como el canal de comunicaciones que tiene seleccionado.

El área Link 11 muestra el estado de los equipos de Link 11 que tenga instalados la unidad (hasta 2). Los leds miniatura reflejan la activación de los modos de red (RC: Roll Call, Br: Broadcast, Sh: Short Broadcast) y de los modos de operación (A: Activo, S: Silencio, L: Listen). El Led NCS indica, para cada red, si la unidad es NCS o PU.

El área Link 16 muestra el estado del equipo de Link 16. Los leds miniatura reflejan el papel de la unidad en la red Link 16, indicando si es FJU Primario (Primer led encendido), FJU secundario (segundo led encendido) o no es FJU (ambos apagados). También indica si el modo de transmisión es pasivo o normal.

El área de mensajes muestra el estado de las cuatro redes. El led correspondiente a la red activa (para enviar) está encendido (verde brillante), y el led correspondiente a la red por la que se recibió el último mensaje está intermitente.

El área de fonía muestra el estado de las tres redes seleccionadas. El led correspondiente a la red activa (para hablar) está encendido (verde brillante), y el led correspondiente a la red por la que alguien habló por última vez está intermitente. En los leds pequeños se indica el altavoz seleccionado para escucha de cada red.


Submarinos

El área Satélite (SAT) muestra el estado del transceptor satelital.

En el área Boyas de Comunicaciones (BOYAS) se indica el tipo de boya de comunicaciones desplegado, y el estado de las boyas (verde encendido: si quedan boyas disponibles en pañol o desplegadas; amarillo: si está desplegando alguna boya; verde apagado: si no quedan boyas disponibles).
