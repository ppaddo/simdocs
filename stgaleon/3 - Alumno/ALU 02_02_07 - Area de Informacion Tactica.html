    A continuación se describe el contenido del Área de Datos del contacto / unidad en Hook. Para incluir la mayor cantidad de información en esta área se emplean abreviaturas que se detallan más adelante. La información que se muestra en esta área depende de si la consola es de alumno o instructor y del elemento seleccionado con el Hook:

    Instructor seleccionando unidades o Alumno seleccionando Unidad Propia:

    CATE BAND AMID Categoría / Bando / Ampliación Identidad
    TIPO NUML CLASE Tipo / Numeral / Clase
    NUM XXXXX RUM 0º Numeral / Rumbo
    VEL 0.0 ALT 0 Velocidad / Altura

    Instructor seleccionando contactos manuales:

    CATE IDEN AMID Categoría / Identidad / Ampliación Identidad
    TIPO NUML Tipo / Numeral
    NUM XXXXX RUM 0º Numeral / Rumbo
    VEL 0.0 ALT 0 Velocidad / Altura

    Alumno seleccionando un contacto adquirido o manual:

    CATE IDEN AM_ID Categoría / Identidad / Ampliación Identidad
    TIPO Tipo
    TN XXXXX ALT 0 Numeral / Altura del contacto
    RUM 0º VEL 0.0 Rumbo / Velocidad del contacto
    DEM 0º DIS 0.0 Demora / Distancia del contacto
    MODE1 00 MODE3 00 Códigos IFF Modo 1 y Modo 3 (vacío si manual)
    MODE2 0000 MODE4 S/N Códigos IFF Modo 2 y Modo 4 (vacío si manual)
    MODEC 0000 Código IFF Modo C (vacío si manual)
    F.O. ORDEN RESPUESTA Force Order: Orden / Respuesta a la Orden
    RR XXXXX HORA V 00:00:00 Report Responsability y Hora de Validez

    Alumno seleccionando una marcación adquirida:

    TN XXXXX SENSOR XXX Numeral / Sensor: ESM, SA, SP
    DEM 0º CAT XXX Demora / Categoría de la marcación
    FREQ 0000.00 BAND XXX Frecuencia (*) / Bando (*)
    PW 00.00 PW (*))
    PRI 0000.00 PRI (*)
    ARP 00 C ARP y Tipo de Radar (*)
    SPOT 00000 SPOT (*)
    HORA V 00:00:00 Hora de Validez

    (*) Los campos FREQ, BAND, PW, PRI, ARP y SPOT no se presentan para detecciones sonar. El Tipo de Radar se indica con las siguientes abreviaturas: Lock-on: L; Sectorial: S; Circular: C; Sectorial y Variable: SV; Circular y Variable: CV.
