Operación

El Simulador Táctico Galen consta de las siguientes aplicaciones software:

Servidor de Ejercicios: aplicación que permite al administrador cargar para su ejecución en red varios ejercicios simultáneos.

Servidor de Modelos: aplicación que se encarga la evaluación modelos en ejecuciones en red.

Preparación: aplicación que facilita el acceso a la base de datos del sistema para configurar equipos, unidades, escenarios y ejercicios.

Consola: aplicación que proporciona el acceso al operador, tanto para alumnos como para instructores (según la asignación de consolas realizada en la preparación del ejercicio) durante las ejecuciones en red. También puede ser empleada para ejecutar ejercicios en local. Por ultimo esta aplicación permite cargar ejercicios grabados, proporcionando también una herramienta para el análisis.

En resumen, independientemente de la aplicación empleada, las herramientas con las que cuenta el operador del Simulador Táctico Galen para las tres fases de la simulación son las siguientes:

  • Herramienta de Preparación.
  • Consola de Alumno.
  • Consola de Instructor.
  • Herramienta de Análisis.
